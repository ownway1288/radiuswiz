# Description
    npm package for adding border-radius to your elements.
# Installation
    `npm i radiuswiz --save`
    ```
    import {radiuswiz} from 'radiuswiz';
    radiuswiz({
        circle: false
    })

    ```
## Options
    radiuswiz supports only one option:
    - _circle_ - _boolean_ (Defaults to false)
